Las condicionales en ingles

  • Zero conditional

    existe una condicion para que se produsca una consecuencia
    formula: if + presente + presente

  • First conditional

    existe una condicion para que se produsca una consecuencia
    la consecuencia siempre ira en futura (will) ya que siempre ocurre despues de la condicion
    formula: if + presente + will + infinitivo.

  • Second conditional

    existe una condicion para que se produsca una consecuencia
    siempre que usemos el verbo to be en el condicional empleamos la fomra de were, para todas las personas. es la fomra que tiene el igles de dar a entender que es algo hipotetico.
    formula: if + past simple + would + infinitivo

  • Third conditional

    existe una condicion para que se produsca una consecuencia
    se usa para expresar resultados hipoteticos de una situacion pasada
    formula: if + past perfect + would + have + participio.
